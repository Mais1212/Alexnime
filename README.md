# Парсер книг с сайта yummyanime.club

Парсер собирает данные с  [сайта](https://yummyanime.clubhttp://tululu.org) и сохранает в файл "Результат.xlsx"

### Как установить

Python3 должен быть уже установлен. Затем используйте pip (или pip3, есть конфликт с Python2) для установки зависимостей:
```
pip install -r requirements.txt
```

### Аргументы

Для удобства пользователя в коде есть аргументы, их нужно указывать при запуске скрипта после ' python main.py '

Аргументы:

```-sp```, ```--start_page``` - Какой страницей начать (По умолчанию стоит 0)

```-ep```, ```--end_page``` - Какой страницей закончить (По умолчанию стоит 2)
